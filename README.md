# Netflix Titles Data Cleaning (Pandas)
## Files
- `code for cleaning the data using pandas` – Python script that performs all data‑cleaning steps.[1]
- `netflix_titles.csv` – cleaned Netflix titles dataset generated by the script.   
- `output.spydata` – Spyder workspace session containing variables after running the code.[3]
- `Screenshot 2025-12-08 214242.png` – screenshot showing the DataFrame output after cleaning.[4]

## Data‑Cleaning Steps
- Loads the raw Netflix dataset into a pandas DataFrame.[5]
- Identifies missing values with `.isnull().sum()` and fills key columns (`country`, `date_added`, `director`, `cast`) with suitable defaults.[1]
- Removes duplicate rows using `.drop_duplicates()`.[6]
- Standardizes text (trims spaces, normalizes casing) for columns such as `country` and `type`.[7]
- Converts `date_added` to `datetime`, adds a `dd-mm-yyyy` formatted column, and enforces proper data types like integer for `release_year`.[8][6]
- Exports the cleaned DataFrame to `netflix_titles.csv` in the working directory.[9][10]
